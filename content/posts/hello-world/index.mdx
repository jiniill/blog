## 블로그를 시작합니다

안녕하세요! 이 블로그의 첫 번째 글입니다.

개발하면서 배운 것들, 생각한 것들을 이곳에 기록할 예정입니다.

### 이 블로그의 기술 스택

- **Next.js 16** - App Router 기반
- **Tailwind CSS v4** - 스타일링
- **Velite** - MDX 콘텐츠 처리
- **Keystatic** - 웹 기반 글쓰기
- **Vercel** - 배포

### 코드 블록 예시

```typescript
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet("방문자"));
```

앞으로 많은 글을 작성할 예정이니 자주 방문해주세요!
